# TaskFlow

## Краткое описание
TaskFlow — это мобильное приложение для планирования и учета задач с поддержкой категорий и уровней важности. 
Оно помогает структурировать ежедневные обязанности и предоставляет полезные советы по продуктивности.

## Основные функции
1.  **Создание задач:** Пользователь может добавлять задачи, указывая заголовок, категорию, приоритет и дополнительную информацию (опционально).
2.  **Обзор списка задач:** Все задачи выводятся в структурированном виде с группировкой по выбранным категориям.
3.  **Локальное хранение:** Все данные сохраняются в памяти устройства.
4.  **Советы по продуктивности:** На главном экране в специальном блоке показывается совет по продуктивности, полученная из открытого внешнего источника (API).

## Структура экранов

### Экран 1: Главный экран (Home Screen)
*   **Заголовок:** "Мои задачи"
*   **Виджет совета:** Динамический виджет, отображающий случайный совет по продуктивности.
*   **Список категорий:** Группировка задач по категориям (работа, отдых, обучение, быт).
*   **Элементы задач:** Для каждой задачи отображается название, категория, приоритет и статус выполнения.
*   **Кнопка "Добавить задачу":** Открывает экран для создания новой задачи.
*   **Кнопка "Статистика":** Переводит пользователя на экран со статистикой.

### Экран 2: Экран добавления задачи (Add Task Screen)
*   **Заголовок:** "Новая задача"
*   **Поле названия:** Поле для ввода названия задачи.
*   **Выбор категории:** Выпадающий список для привязки задачи к определенной категории.
*   **Выбор приоритета:** Сегментированный выбор уровня приоритета (высокий, средний, низкий).
*   **Поле описания:** Необязательное текстовое поле для деталей и комментариев.
*   **Кнопка "Сохранить":** Сохраняет новую задачу в общий список.
*   **Кнопка "Назад":** Возвращает пользователя на главный экран.

### Экран 3: Экран статистики (Statistics Screen)
*   **Заголовок:** "Статистика"
*   **Общая статистика:** Отображается общее число задач и процент выполнения.
*   **Статистика по категориям:** Прогресс выполнения для каждой категории.
*   **Кнопка "Назад":** Возвращает пользователя на главный экран.

## Навигация
Главный экран (Home Screen) <--> Экран добавления задачи (Add Task Screen)
Главный экран (Home Screen) <--> Экран статистики (Statistics Screen)

Навигация реализуется с помощью простого `Navigator` в Flutter. 
Кнопка "Добавить задачу" на главном экране осуществляет переход на экран добавления задачи.
Кнопка "Статистика" открывает экран со статистикой. 
Кнопки "Назад" (или стрелки в AppBar) возвращают пользователя на главный экран.
